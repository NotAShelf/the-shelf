# the-shelf

> **Extra utilities to keep code consistent and non-repetitive.**

## Usage

```js
// ESM
import utils from 'the-shelf';

// CommonJS
const utils = require('the-shelf');
```

## Installation

```sh
$ npm i https://github.com/NotAShelf/the-shelf --save
```
